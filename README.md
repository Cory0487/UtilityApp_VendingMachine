# UtilityApp Vending Machine (C++)

## Description
This project is a console-based vending machine application written in C++.  
It simulates the basic functionality of a real vending machine, allowing users to:
- View available items
- Select products using item codes
- Add items to a basket
- Check out using a fixed Â£20 payment
- Receive correct change

## Features
- Menu-driven console interface
- Product selection by code
- Basket management
- Input validation
- Automatic change calculation
- Clear dispensing messages

## How to Compile and Run
Open a terminal in the project folder and run:

g++ main.cpp -o vending  
./vending

## Example Usage
1. Show items  
2. Add item (enter code and quantity)  
3. View basket  
4. Checkout  
5. Exit
